 - USE + bata base (denomina qual será o banco de dados que será utilizado para rodas os comandos).
 - SP_HELPDB + Banco de dados (trás informações sobre o banco dedados escolhido).
 - "--" (Comentário, para informações sobre o seu código ou da linha do comando).
 - CREATE TABLE + tabela (usa-se paa criar uma tabela). 
 - PRIMARY KEY (Chave Primaria de uma tabela).
 - FOREIGN KEY (chave estrangeira, onde tem uma coluna de referência da PRIMARY KEY).
 - IDENTITY (gera um número único sempre que um novo registro é criado na tabela).
 - CHECK (limita os valores que podem ser colocados em uma coluna).
 - DEFAULT (para inserir um valor padrão em uma coluna).
 - SELECT (pode selecionar todos(*) ou as colunas que você deseja fazer a sua pesquisa).
 - FROM + tabela (seleciona de qual tabela o usuário deseja puxar a informação).
 - WHERE (condição para detalhar mais a sua pesquisa).
 - GROUP BY (condição de agrupamento das informações que deseja puxar de uma tabela).
 - INSERT (iserir valores em uma tabela).
 - VALUES (valor que vai ser inserido na coluna da tabela).
 - ALTER TABLE + tabela (alterar uma tabela ou uma coluna).
 - DROP TABLE + tabela (usado para deletar colunas ou tabelas).
 - ADD COLUMN + coluna (adiciona uma coluna na tabela especificada).
 - TRUNCATE TABLE + tabela (limpa os registros da tabela sem remove-la).
 - UNION (Para unir dados de duas ou mais tabelas).
 - INTO (insere a consulta que o usuário deseja em uma nova tabela).
